# Trivia-Web-App
Web app utilizing api's that allows user to take part in a trivia game. User can save score for competition against other users!
# Description
* Movie Trivia  is an Game Website that allow user to Test their Movie knowledge with a series if questions.
 
 * Questions from All decades of movies tv shows and more, Along with Funny Tech Jokes for extra Humor.

# Goals

* Connecting a the Api, Allowing smooth transtions with questions And jokes. Having a user friendly look and feel to the app.
